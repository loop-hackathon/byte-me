# CloudHelm Overview

## ✅ Working Features
* **Cost Radar:** Multi-cloud cost tracking, ML anomaly detection, budgeting, and forecasting.
* **Resource Efficiency:** Compute/storage tracking, underutilization identification, and waste scoring.
* **Application Health:** End-to-end performance and latency monitoring with Docker/K8s integration.
* **Incident Management:** AI-powered incident summaries and timeline tracking.
* **Release Impact Analysis:** Deployment risk scoring and GitHub release correlation.
* **CloudHelm Assistant:** Mistral AI CLI coding assistant `/test`, `/lint`, `/run` inside the app.

## ⚠️ Known Issues & Limitations
* **No explicit bugs found in the codebase.**
* **Free-Tier Limits:** Prone to cold starts, database limits, and rate limits.
* **Port Conflicts:** Local dev requires strictly ports `8000` & `5173` to be free.
* **AI Dependency:** Core AI features require Gemini and Mistral API keys to function.
